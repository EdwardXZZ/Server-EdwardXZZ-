## 🚀 ¿Qué es Saki Images?

**Saki Images** es un subidor de archivos gratuito creado por **Edward**. Permite subir imágenes, videos y audio con un límite de **100 MB** por archivo. Los archivos duran **60 días** antes de ser eliminados automáticamente.

---

## ✨ Características

| Función | Descripción |
|---------|-------------|
| 🖼️ Imágenes | PNG, JPG, GIF, WEBP, SVG, BMP, ICO |
| 🎬 Videos | MP4, WEBM, MOV, AVI, MKV, FLV |
| 🎵 Audio | MP3, WAV, OGG, M4A, FLAC, AAC |
| 📏 Límite | 100 MB por archivo |
| ⏳ Duración | 60 días (se borran automáticamente) |
| 📋 Copiar URL | Un clic para copiar el enlace |
| 🔗 Abrir | Vista previa directa |
| 🗑 Eliminar | Borrar archivos manualmente |
| 📊 Barra de progreso | Animación de subida en tiempo real |
| 🧹 Limpieza automática | Cada 24h se borran los expirados |
| 📱 Responsive | Funciona en móvil, tablet y PC |

---

## 🛠️ Tecnologías

- **Node.js** + Express
- **Multer** para subida de archivos
- **node-cron** para limpieza automática
- **UUID** para nombres únicos
- **Render** para hosting
